Conoce los componentes que hacen funcionar a SIAR

Hardware

Llaveros RFID

Llaveros RFID para colocar en los productos, cada uno con un identificador único.

Lector RFID

Módulo lector RFID (como el RC522) compatible con Arduino para detectar las etiquetas RFID.

Arduino Uno

Microcontrolador para procesar la información del lector RFID y comunicarse con otros dispositivos.

Buzzer activo

Para generar alertas locales inmediatas cuando se detecte un producto no autorizado.

Módulo de comunicación

ESP8266 (Wifi) o HC-05/HC-06 (Bluetooth) para enviar los datos desde el Arduino a la app móvil o al servidor.

Fuente de alimentación

Adaptador o batería para alimentar el Arduino y sus componentes.

Cables y protoboard

Para realizar las conexiones entre el Arduino, el lector RFID y otros módulos.

Smartphone

Dispositivo móvil con la aplicación instalada para recibir alertas y gestionar los datos.

Sensor ultrasónico HC-SRO4

Será utilizado para medir la distancia del equipo de trabajo.

Módulo relay - Relé 1 con Diodo 5V

Este módulo se activa con 5V de corriente continua para activar la corriente alterna.

Software

Arduino IDE

Para programar el código del Arduino, gestionando el módulo RFID y la comunicación con la app.

Bibliotecas RFID

Librerías como MFRC522.h para controlar el lector RFID y leer etiquetas.

Flutter

Framework multiplataforma para desarrollar la aplicación móvil en Android.

Firebase

Base de datos en la nube para almacenar los datos en tiempo real.

Web (Python/Django)

Framework web del lenguaje de programación Python para el desarrollo de la página web informativa.

App Móvil

Aplicación móvil disponible para el sistema operativo Android.
