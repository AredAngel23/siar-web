Componentes del Sistema

Visualiza todos los componentes hardware y software que conforman nuestro sistema.

Componentes IoT

Frontend (Hardware)

Llaveros RFID

Dispositivos pasivos que contienen un chip y una antena para almacenar y transmitir datos de identificación de forma inalámbrica cuando se acercan a un lector RFID.

Lector RFID

Dispositivo que emite señales de radio para detectar y leer la información almacenada en los tags RFID cuando estos se encuentran dentro de su rango de operación.

Sensor ultrasónico HC-SRO4

Sensor que utiliza ondas ultrasónicas para medir distancias. Emite pulsos y mide el tiempo que tarda el eco en regresar, permitiendo calcular la distancia a objetos.

Buzzer Activo

Componente electrónico que genera sonidos cuando recibe una señal eléctrica. Utilizado para alertas audibles en el sistema.

Módulo Relay - Relé 1 con Diodo 5V

Interruptor controlado eléctricamente que permite activar o desactivar circuitos de mayor potencia mediante señales de bajo voltaje desde el microcontrolador.

Fuente de Alimentación

Proporciona la energía eléctrica necesaria para el funcionamiento de todos los componentes del sistema, con el voltaje y amperaje adecuados.

Cables y Protoboard

Elementos para la conexión física entre componentes. La protoboard permite crear circuitos sin soldadura, mientras que los cables establecen las conexiones eléctricas.

Backend (Software)

Arduino Uno

Microcontrolador que actúa como el cerebro del sistema IoT, procesando las señales de los sensores y controlando los actuadores según la lógica programada.

Arduino IDE

Entorno de desarrollo integrado utilizado para escribir, compilar y cargar el código en la placa Arduino. Facilita la programación del microcontrolador.

Bibliotecas RFID

Conjunto de funciones y clases predefinidas que simplifican la interacción con los lectores RFID, permitiendo leer y procesar la información de los tags de manera eficiente.

Desarrollo Web

Frontend

Interfaz Web

Interfaz de usuario desarrollada con HTML, CSS y JavaScript que permite a los usuarios interactuar con el sistema a través de un navegador web.

Backend

Python/Django

Framework de desarrollo web en Python que proporciona una estructura robusta para crear aplicaciones web. Gestiona la lógica del servidor, la autenticación y la interacción con la base de datos.

Firebase (Web)

Plataforma de desarrollo que proporciona servicios como base de datos en tiempo real, autenticación de usuarios y almacenamiento en la nube para la aplicación web.

Desarrollo Móvil

Frontend

Flutter

Framework de UI de Google para crear aplicaciones nativas para móvil, web y escritorio desde una única base de código. Proporciona widgets personalizables para crear interfaces de usuario atractivas.

App Móvil

Aplicación para dispositivos móviles que permite a los usuarios interactuar con el sistema IoT de forma remota, recibir notificaciones y controlar dispositivos.

Backend

Firebase

Proporciona servicios backend para la aplicación móvil, incluyendo base de datos en tiempo real, autenticación, notificaciones push y análisis de uso.
